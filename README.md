### Otomat Makinesi Arayüzü

| Kod | Ürün     | Fiyat (TL) |
| --- | -------- | ---------- |
| A1  | Çikolata | 12.50      |
| A2  | Kola     | 20.00      |
| A3  | Cips     | 15.75      |
| A4  | Su       | 8.00       |

#### Kullanım:

- Otomatı seçin.
- Kodu girerek ürünü seçin.
- Ödeme yapın. (Abonman Kart / Kredi Kartı)
- Ürünü alın.
